Gradasi Warna Tulisan Menggunakan CSS

Seperti yang kita ketahui, dalam pembuatan website menggunakan HTML, kita dapat merubah warna background semua element yang memiliki panjang dan lebar. Tentu, kita juga dapat menciptakan unsur gradasi pada element tersebut.

Membuat gradasi pada element menggunakan CSS sangatlah mudah, kita hanya perlu menambahkan property linear-gradient() pada property background 

selector {
background : linear-gradient(to right,#00a8a8, #006464);
};

Lalu, bagaimana dengan tulisan ? Bisakah kita membuat unsur warna gradasi pada tulisan seperti yang biasa kita buat pada element ?

Tentu saja bisa.

Nah, kali ini, saya akan membagikan bagaimana cara membuat gradasi warna tulisan menggunakan CSS

Text pada halaman HTML adalah string yang berada didalam suatu tag

"Hidup itu sederhana, kita yang membuatnya sulit."

- Confucius

Seperti contoh diatas, kita bisa dengan mudah memberikan warna background bergradasi pada element tag dalam HTML

Namun, berbeda dengan tulisan. Ada beberapa property tambahan yang harus kita tulis jika kita ingin membuat tulisan memiliki gradasi warna

Buka text editor kamu, lalu buatlah file index.html baru dan style.css baru

Buka file index.html dan beri sintaks dasar HTML

<!DOCTYPE html>
    <html lang="en">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <link rel="stylesheet" href="style.css">
        <title>Mengubah Scrollbar</title>
    </head>
    <body>
                                    
    </body>
    </html>

Setelah itu, buatlah satu element h1, lalu tambahkan text sembarang atau apapun kedalamnya 

<!DOCTYPE html>
    <html lang="en">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <link rel="stylesheet" href="style.css">
        <title>Mengubah Scrollbar</title>
    </head>
    <body>
        <h1>Selamat datang di website Azrilardian</h1>
    </body>
    </html>

Selesai, lalu bukalah file style.css yang sudah kamu buat tadi. Biasakan untuk menggunakan external CSS agar tidak mencampur adukkan antara HTML dan CSS.

Pastikan index.html dan style.css sudah kamu hubungkan.

Seleksi element h1, lalu tambahkan value berikut pada property background, atau ubah sesuai selera

h1{
background : linear-gradient(to right,#ff6a6a, #353434);
};

Kemudian, tambahkan property dan value berikut

h1{
background : linear-gradient(to right,#ff6a6a, #353434)
-webkit-background-clip : text;
-webkit-text-fill-color : transparent;
};

Hasil

Nah, sebenarnya inti dari membuat gradasi warna tulisan menggunakan CSS adalah pada kedua property dan value tersebut

Property dan value -webkit-background-clip : text; berfungsi untuk memasukan warna background ke dalam tulisan.

Dan property dan value -webkit-text-fill-color : transparent; berfungsi untuk menjadikan warna background pada tag h1 menjadi transparent.
